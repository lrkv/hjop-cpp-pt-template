hJOP PT server C++ client template
==================================

## Dependencies

* <https://github.com/nlohmann/json>
* <https://github.com/yhirose/cpp-httplib>
* <https://github.com/tobiaslocker/base64>

All dependencies are included in this repository as submodules.

## Build

```bash
$ mkdir build
$ cd build
$ cmake ..
$ make
```
